クラウド接続設定
1
Firebase プロジェクトを作成
console.firebase.google.com → 「プロジェクトを作成」
2
Firestore Database を有効化
左メニュー「Firestore Database」→「データベースを作成」→ 本番環境モード
3
セキュリティルールを設定
Firestore「ルール」タブ → allow read, write: if true; に変更して「公開」
4
設定情報をコピーして貼り付け
⚙設定 → 全般 → ウェブアプリを追加（</>） → firebaseConfig の { } をコピー
データはご自身のFirebaseプロジェクトに保存されます。無料で使えます。

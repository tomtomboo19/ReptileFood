クラウド接続設定
1
Firebase プロジェクトを作成
console.firebase.google.com を開いて「プロジェクトを作成」→ プロジェクト名（例: reptifeed）を入力 → 作成完了
2
Firestore Database を有効化
左メニュー「Firestore Database」→「データベースを作成」→ 本番環境モード → 「次へ」→「有効にする」
3
セキュリティルールを設定
Firestore の「ルール」タブ → 以下に書き換えて「公開」
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
   match /{doc=**} { allow read, write: if true; }
  }
}
4
設定情報をコピー
プロジェクトの「設定（⚙）」→「全般」→ 下にスクロール →「ウェブアプリを追加（</>）」→ アプリ登録 → 表示された firebaseConfig の { } の中身ごとコピーして下に貼り付け
データはご自身のFirebaseプロジェクトに保存されます。
無料プランで十分使用できます。
